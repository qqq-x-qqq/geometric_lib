# Geometric Library - Документация

## Общее описание решения

Geometric Library - это библиотека на Python для вычисления площадей и периметров различных геометрических фигур.

### Поддерживаемые фигуры:
- Круг
- Квадрат  
- Прямоугольник
- Треугольник

## Модуль circle.py

### Функция `area(r)`
Вычисляет площадь круга по его радиусу. Формула: S = πR²

**Параметры:**
- `r` (int | float): радиус круга

**Возвращает:**
- float: площадь круга

**Пример:**
```python
area(5)  # возвращает 78.53981633974483
```

### Функция `perimeter(r)`
Вычисляет длину окружности по её радиусу. Формула: P = 2πR

**Параметры:**
- `r` (int | float): радиус окружности

**Возвращает:**
- float: длина окружности

**Пример:**
```python
perimeter(5)  # возвращает 31.41592653589793
```

## Модуль rectangle.py

### Функция `area(a, b)`
Вычисляет площадь прямоугольника по длинам его сторон.

**Параметры:**
- `a` (int | float): длина первой стороны
- `b` (int | float): длина второй стороны

**Возвращает:**
- int | float: площадь прямоугольника

**Пример:**
```python
area(4, 5)  # возвращает 20
```

### Функция `perimeter(a, b)`
Вычисляет периметр прямоугольника по длинам его сторон.

**Параметры:**
- `a` (int | float): длина первой стороны
- `b` (int | float): длина второй стороны

**Возвращает:**
- int | float: периметр прямоугольника

**Пример:**
```python
perimeter(4, 5)  # возвращает 18
```

## Модуль square.py

### Функция `area(a)`
Вычисляет площадь квадрата по длине его стороны. Формула: S = a²

**Параметры:**
- `a` (int | float): длина стороны квадрата

**Возвращает:**
- int | float: площадь квадрата

**Пример:**
```python
area(5)  # возвращает 25
```

### Функция `perimeter(a)`
Вычисляет периметр квадрата по длине его стороны. Формула: P = 4a

**Параметры:**
- `a` (int | float): длина стороны квадрата

**Возвращает:**
- int | float: периметр квадрата

**Пример:**
```python
perimeter(5)  # возвращает 20
```

## Модуль triangle.py

### Функция `area(a, h)`
Вычисляет площадь треугольника по основанию и высоте.

**Параметры:**
- `a` (int | float): длина основания треугольника
- `h` (int | float): высота, проведённая к основанию

**Возвращает:**
- int | float: площадь треугольника

**Пример:**
```python
area(10, 5)  # возвращает 25
```

### Функция `perimeter(a, b, c)`
Вычисляет периметр треугольника по длинам его сторон.

**Параметры:**
- `a` (int | float): первая сторона
- `b` (int | float): вторая сторона  
- `c` (int | float): третья сторона

**Возвращает:**
- int | float: периметр треугольника

**Пример:**
```python
perimeter(3, 4, 5)  # возвращает 12
```
